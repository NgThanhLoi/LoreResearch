# Character Audit Backlog

## Purpose

Track which characters require audit and which phase they belong to.

## Audit Levels

### Level 1 — Inventory Only
Character has been listed and risk-classified. No deep claim audit yet.

### Level 2 — OOC Risk Audit
Character has core canon traits, headcanon risks, and OOC traps identified.

### Level 3 — Full Character Page
Character has enough source support to become a stable wiki entity page.
