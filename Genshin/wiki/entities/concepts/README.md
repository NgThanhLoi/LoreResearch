# Concepts Index

## Purpose

* Quản lý hồ sơ các khái niệm trừu tượng, thế giới quan, quy luật vũ trụ hoặc truyền thuyết cosmological của Genshin Impact (Bầu trời giả, Kẻ Giáng Lâm, Tri-Lunar Authority, Vực Sâu, v.v.).
* Hạn chế rủi ro người viết tự bịa các cơ chế vũ trụ học làm phá vỡ logic game.

## Page Schema Summary

Mọi hồ sơ khái niệm phải tuân theo cấu trúc mẫu tại `_template.md` bao gồm:
* **Definition:** Định nghĩa cốt lõi của khái niệm.
* **Canon Usage vs Theory Usage:** Cách dùng đã xác thực vs suy đoán cộng đồng.
* **Related Powers / Events / Entities:** Các liên kết chéo.
* **Fanfic Risk:** Rủi ro khi đưa vào cốt truyện.

## Current Status & Priority List

* **Trạng thái:** Các khái niệm trừu tượng được quản lý qua `unverified-lore-register.md` và `canon-lock.md` của planning.
* **Độ ưu tiên hoàn thiện:**
  1. **Bầu trời giả (False Sky):** Quy luật vũ trụ học quan trọng.
  2. **Kẻ Giáng Lâm (Descenders):** Bản chất của các thực thể ngoài thế giới.
  3. **Kuuvahki (Sức mạnh mặt trăng):** Định dạng năng lượng cho vùng Nod-Krai.

## How to Add New Page

1. Nhân bản file `_template.md` sang tên mới (ví dụ: `False_Sky.md`).
2. Điền frontmatter YAML và ghi chú rõ ràng các rủi ro cốt truyện (fanfic risks).
