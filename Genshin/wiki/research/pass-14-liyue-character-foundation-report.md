## Source Policy Used

- **Primary sources:** `sources/voice_lines/characters/[Name].md` — used for identity, personality traits, relationships, voice guide, and broad role.
- **Indirect sources:** `entities/character_intelligence/[Name].md` — used ONLY as writing guidance for OOC risks, strongly implied traits, and theory identification. NOT treated as primary canon.
- **Supporting context:** `wiki/entities/characters/character-inventory.md`, Pass 11–13 roadmap and governance files.
- **Not used:** External web, new source IDs, character-story files, quest dialogue transcripts, API data.
